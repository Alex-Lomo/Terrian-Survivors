# Survivors-Like Unity Game

A 2D Survivors-Like game developed in Unity, inspired by titles such as *Vampire Survivors*.  
This project explores gameplay architecture, stat systems, weapon mechanics, enemy waves, and scene management in Unity.

## Features
- Multiple playable characters with unique stats, growth patterns, and starting weapons
- Full item system: weapons, accessories, evolutions, and level-up options
- Several weapon types: projectile, aura, and special effect weapons
- Enemy wave system with bosses and a 10-minute round structure
- Experience, leveling, and stat-scaling mechanics
- Collectibles such as XP gems and healing items
- Multiple scenes: Title Screen, Character Selection, Gameplay
- Custom UI: level-up menu, pause screen, game over summary, inventory display

## How to Play
- **WASD** — move your character  
- Weapons attack automatically  
- Collect XP gems to level up  
- Choose upgrades and survive as long as possible  
- Fight through waves of enemies and bosses

## Project Structure
Assets/
├─ Scripts/ # Gameplay logic (player, enemies, weapons, items, UI)
├─ Art/ # Sprites, animations, VFX
├─ Prefabs/ # Characters, enemies, weapons, collectibles
└─ Scenes/ # Title, Character Select, Gameplay

## Requirements
- Unity **2022.x** or newer  
- Target platform: **PC (Windows)**
